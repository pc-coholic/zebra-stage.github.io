These parms are used to enter the conditions under which a key-code will be sent as the Key Behavior in various Key State Mapping Tables when a double-press of any scan trigger is detected.

Parm names:

  • For Base Key State Table: DoubleTriggerKeyCodeAdvanced
  • For Shift Key State Table: DoubleTriggerKeyCodeShiftState
  • For Alt Key State Table: DoubleTriggerKeyCodeAltState
  • For Ctrl Key State Table: DoubleTriggerKeyCodeCtrlState
  • For Fn Key State Table: DoubleTriggerKeyCodeFnState

Parm value input rules:

  • String containing a valid Key Code
