Scoped Storage Guide

Enterprise Browser 3.3

UNdeR ConsTRuCtioN

Overview

As a part of playstore compliance, Enterprise Browser now targets Android API level 30, which enforces the Android’s scope storage restrictions on Android 11. Android’s Scoped Storage on A11 sets several restrictions on applications.

Mass deployments to sdcard is no more supported.

In the older Enterprise Browser versions, Admins had a choice to copy the Enterprise Browser configuration files to the SD Card root directory or to enterprisebrowser installation directory. With the new Enterprise Browser, the config file must be deployed to the Enterprise partition </enterprise/device/enterprisebrowser/TechDocs>.

Enterprise Browser application using camera database, File, printer APIs etc., to read/write from sdcard is no longer supported.

Note: Though Scoped Storage restriction affects only A11 (and above) devices, Zebra recommends the following solutions for A10 and A11 when EB3.3 is mixed deployed on both A10 and A11.
