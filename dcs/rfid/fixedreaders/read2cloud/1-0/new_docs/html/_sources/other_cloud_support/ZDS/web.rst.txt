====================================
Configure device using Web Interface
====================================

To enroll the reader to Zebra Data Services, follow the steps below:

1. Open a web browser to connect to the reader using the host name or IP address.

3. Click Communication > Zebra IoT Connector > Configuration page.

.. image:: images/ZDS_w1.png

4. Click on ``Add Endpoint`` and select ``ZDS`` for ``Endpoint Type``.

5. Fill ``Endpoint Name`` and ``Endpoint Description`` values.

6. The following parameters are required to enroll the reader. Values for these parameters are obtained after creating and registering your Cloud application and need to be keyed into this page.
    
    a. Claim Code - Secret code for enrolling the device (:ref:`ZDS Setup`).

.. image:: images/ZDS_w2.png

7. Click on ``Add``.

8. Once added the endpoint will be seen under ``Endpoint Configurations``.

9. Under Interface Configuration select ZDS for ``Management Interface``, ``Control Interface``, and ``Data Interface``. ZDS is not supported for ``Management Events Interface``.

10. Click on ``update``.

.. image:: images/ZDS_w3.png

11. Navigate to Communication > Zebra IoT Connector > Connection page to view current connection status.

12. Click on ``Connect``.

13. Check ``Connection Status`` section to view the current connection status.

.. image:: images/ZDS_w4.png
