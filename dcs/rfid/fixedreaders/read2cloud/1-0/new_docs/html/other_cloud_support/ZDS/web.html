Configure device using Web InterfaceÂ¶

To enroll the reader to Zebra Data Services, follow the steps below:

  1. Open a web browser to connect to the reader using the host name or IP address.

  1. Click Communication > Zebra IoT Connector > Configuration page.

  1. Click on Add Endpoint and select ZDS for Endpoint Type.

  2. Fill Endpoint Name and Endpoint Description values.

  3. The following parameters are required to enroll the reader. Values for these parameters are obtained after creating and registering your Cloud application and need to be keyed into this page.

    1. Claim Code - Secret code for enrolling the device (Enrolling the Reader).

  1. Click on Add.

  2. Once added the endpoint will be seen under Endpoint Configurations.

  3. Under Interface Configuration select ZDS for Management Interface, Control Interface, and Data Interface. ZDS is not supported for Management Events Interface.

  4. Click on update.

  1. Navigate to Communication > Zebra IoT Connector > Connection page to view current connection status.

  2. Click on Connect.

  3. Check Connection Status section to view the current connection status.
